Welcome !

Zorka Intranet Collector collects performance data from Zorka agents across the network. You can view details about execution of all instrumented applications here.

Tips

  • to display trace list from host of interest, double click host from the list from left panel;
  • to display more details about trace, click that small arrow icon on the left;
  • to display call method tree of a trace double click on specific trace in trace list;
  • to search among traces, type some text (or regexp starting with '~') in the input box above and press ENTER or click 'Search' button;
  • you can also use SQL-like queries; click 'QL' button to enable it;
  • in order to enable searching full call tree (not only data visible in top level trace list), click 'Deep Search' button;
